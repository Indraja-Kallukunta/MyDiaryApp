# MyDiaryApp

Developed a web-based diary application to securely store and manage daily notes

Implemented full CRUD operations (Create, Read, Update Delete) for diary entries

Built using Java , JSP, and Spring Boot

Designed user interface using HTML, CSS, and JavaScript

Integrated with Oracle SQL database for data storage

Used Spring Data JPA (Hibernate) for efficient data persistence

Followed MVC architecture to ensure clean code structure
